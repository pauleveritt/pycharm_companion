PyCharm Professional has a lot of support for the various pieces of modern web development. In this series we put several of these pieces and technologies together end-to-end to build a cloud application:

  • Vagrant
  • Ansible
  • Flask
  • virtualenv
  • Ubuntu

The series covers:

  • Developing in a VM with Vagrant and Ansible
  • Building an API using Flask-RESTful, and using the PyCharm HTTP Client
  • Creating a Python Development Environment on Amazon EC2
